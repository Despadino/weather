Архив среднемесячных температур Москвы (2014–2024 гг.) с сайта "Погода и климат":
http://www.pogodaiklimat.ru/history/27612.htm
Инструкция:
1. Установите зависимости: pip install -r requirements.txt
2. Запустите main.py или build/weather.exe
3. Пример данных в data/temperatures.txt:
   - Формат: Год, янв, фев, ..., дек (разделитель - запятая).
   - Пример строки: 2014, -5.3, -7.1, ..., 4.8
